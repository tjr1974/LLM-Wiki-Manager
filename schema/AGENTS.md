# AGENT EXECUTION CONTRACT (AI-FIRST)

Primary outputs are machine artifacts under `ai/` and `normalized/`.
Human-readable markdown is deferred until final synthesis phase.

## Priority order
1. `ai/schema/*.json`
2. `ai/artifacts/*.ndjson` and `ai/runtime/*.json`
3. `normalized/*`
4. `wiki/*` (final-phase projection only)

## Hard rules
- Do not optimize for prose readability.
- Prefer compact keys and deterministic serialization.
- Every claim must carry resolvable evidence ids (`sid:cid`) and confidence (`h|m|l`).
- Never mutate `raw/` files.

## Shared helpers (`scripts/wiki_paths.py`)

- `wiki_source_yaml_id()` resolves ids for **`wiki/sources/*.md`** from front matter (**`source_id`** then **`sid`**), else the file stem (used by **`wiki_compiler`** citation labels).
- `normalized_manifest_sid()` reads **`sid`** then **`source_id`** from **`normalized/<bundle>/manifest.json`**, else the parent directory name (shared by **`wiki_compiler`**, **`validate_wiki.py`**, **`project_sources`**, **`generate_source_wiki`**).
- `domain_targets_schema_path()` resolves the newest **`ai/schema/domain_targets.vN.json`** schema file.
- `validate_wiki_argv_from_env()` splits **`VALIDATE_WIKI_ARGS`** the same way **`Makefile`** **`wiki-validate`** / **`wiki-check`** / **`wiki-ci`** append flags to **`validate_wiki.py`**. **`autopilot.py`** and **`daemon.py`** read **`VALIDATE_WIKI_ARGS`** from the process environment (**`daemon.py`** runs **`autopilot.py --with-queue`** each cycle).
- `autopilot_log_tail_chars(failed=…)` reads **`AUTOPILOT_LOG_TAIL_CHARS`** for stdout/stderr tail sizing shared by **`autopilot.py`** step records and **`daemon.py`** heartbeat JSON (defaults **2000** on success, **16000** on failure when the env var is unset).
- `autopilot_daemon_stderr_tail_chars(failed=…)` floors the **`daemon.py`** heartbeat **`err`** tail using **`AUTOPILOT_DAEMON_STDERR_TAIL_MIN`** (plus **`AUTOPILOT_SOFT_FAILURE_STDERR_NOTICE`** slack) so the soft-failure substring survives long leading **stderr** even when **`AUTOPILOT_LOG_TAIL_CHARS`** is very small (**`stdout`** still uses **`autopilot_log_tail_chars`** only).
- **`AUTOPILOT_SOFT_FAILURE_STDERR_NOTICE`** string shared by **`autopilot.py`** (stderr line when **`soft_failures`** is non-empty while **`ok`** stays true) and **`daemon.py`** (detects the same substring in captured autopilot **stderr** when **`rc`** is **0**).
- **`AUTOPILOT_DAEMON_STDERR_TAIL_MIN`** (``4096``) lower bound for **`daemon.py`** heartbeat **`err`** retention with **`autopilot_daemon_stderr_tail_chars()`** so long leading **stderr** does not push that substring out of the tail when **`AUTOPILOT_LOG_TAIL_CHARS`** is small.
- **`utc_now_iso()`** returns a UTC ISO-8601 timestamp for runtime rollups. When **`SOURCE_DATE_EPOCH`** is set (reproducible-builds convention, Unix seconds), that instant is used instead of wall clock for **`build_claims.py`**, **`build_coverage_matrix.py`**, **`dedupe_runtime.py`**, **`validate_external_links.py`**, **`validate_human_readiness.py`**, and **`validate_ingest_queue_health.py`**.
- **`resolve_repo_root(cli_override)`** and **`safe_repo_rel(path, root)`** support **`--repo-root`** tooling (release export validators, **`build_hub_links`**, **`validate_human_*`**).
- **`human_site_wiki_route.py`** exposes URL path ↔ **`wiki/**/*.md`** identity helpers (**`data-wiki-rel`** on exported HTML vs graph ids without **`.md`**). Used by **`validate_human_site_wiki_rel.py`**, **`build_human_site_discovery.py`**, and **`report_wiki_human_site_coverage.py`**.

## Core pipeline
1. Normalize source -> `normalized/<sid>/manifest.json` + `chunks.ndjson`.
2. Compile runtime index -> `ai/runtime/src.min.json`, `ai/runtime/chunk.min.ndjson`, `ai/runtime/graph.min.json`, `index/links.json`, `ai/runtime/backlinks.min.json`, then `scripts/dedupe_runtime.py` (**`make wiki-compile`** always runs this after `wiki_compiler.py`).
3. Validate evidence integrity before any wiki projection.
4. Query against runtime artifacts and emit JSON-first answers.

For assistants instructed to maximize evidence coverage across **`wiki/sources/`**, **`prompts/wiki-corpus-authoring.txt`** binds the mandated sequence (**`scripts/find_sources_for_topic.py`** is triage-only). Scripted agents still prioritize **`ai/`** and **`normalized/`** outputs over prose polish unless the task is explicitly human-wiki authoring.
